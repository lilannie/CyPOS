Uses:
-------------------------------------------
Install dependencies on local: "npm install"
Run server on localhost: "npm start"
Access website at: "localhost:8080"

Directory Structure:
----------------------------------------
database: DB scripts
files: extra files for testing, reference, etc.
html: extra html files for testing, referencee, etc.
js: general javascript files
lib: external library files
views: public html files that are seen by the user
files in root directory: scripts ran on the server
